About Me

Computer Science graduate and Frontend Developer specializing in React.js and Next.js. Passionate about building responsive web applications, creating clean user interfaces, and developing scalable frontend solutions. Experienced with API integrations, TypeScript, UI/UX design, and modern web technologies, with a continuous focus on learning and growth.

Experience

Frontend Developer

As a Frontend Developer, I have worked on multiple projects using React.js, Next.js, JavaScript, and TypeScript, focusing on building responsive and user-friendly web applications. My experience includes creating reusable UI components, integrating REST APIs, implementing CRUD operations, and optimizing user experience across different devices. I have also worked with modern development tools and libraries such as Redux Toolkit, Axios, TanStack Query, and Tailwind CSS to build scalable frontend solutions.

Desktop Application Development

Alongside frontend development, I have experience as a C# Windows Forms Developer at ICLEB (INFOCENTER - LEBANON), where I developed and maintained multiple desktop applications. My work involved building Windows-based interfaces, handling data operations, and supporting internal business workflows. This experience strengthened my understanding of application structure, logic implementation, and software maintenance.

UI/UX Design

I am also interested in UI/UX design and have been designing web and mobile interfaces using Figma. I focus on creating clean, responsive, and user-centered designs while applying principles such as visual hierarchy, usability, and consistency. My design experience helps me bridge the gap between development and user experience, allowing me to build interfaces that are both functional and visually engaging.
