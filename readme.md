# DecodeLabs Portfolio

A modern and responsive developer portfolio website built to showcase projects, skills, experience, and creative work in a clean and professional way.

## 🚀 Features

- Responsive design for all devices
- Modern UI/UX layout
- Projects showcase section
- Skills and technologies section
- Contact section
- Smooth navigation and clean structure
- Optimized performance

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive Design
- Git & GitHub
